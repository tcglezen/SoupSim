Progress Check

  • Imported project 4 code
    • We decided that since we are going to be using a particle based system that we could use many of the like structures from project 4: cloth simulator. By using mass points and springs, we hope to well simulate fluid dynamics.
  • Planned general code layout and project structure
    • For this project, we decided to use a lot of supporting libraries, many of which can be found in previous projects. (eg. CGL) However, there are many things that we are in the process of developing. For example, given a starting environment, how can we have the water go sploosh sploosh? One idea is that we can attempt to load a json file as well which can contain the position and properties of the water particles. This enables us to quickly create scenes for us to explore.
  • Exploring Berkeley Gfx
    • Progress is not always linear and often what paths we take for development. Currently we are experimenting with the use of Berkeley Gfx shaders. We hope that we end up using Berkeley Gfx in our final project, however only time and hard work will tell.
  • Research Smooth Particle Hydrodynamics
    • We are all smooth brains who really do be out here choosing a fluid simulation based project while knowing fuckall about fluid dynamics. However, the knowledge required to understand fluid dynamics can be quickly overturned by the power of friendship and our desire to learn!
    • As a result, we have been dedicating time into learning how particles in fluid systems should react.
Since we are unable to actually write code and test our implementation until setup is complete, we decided to multitask by creating pseudocode and researching physics. The general pseudocode (without yet implementing viscosity) is as follows:
pseudocode

Initialize all particles

Set t = 0

Choose a ∆t

for i from 0 to n

for j from 1 to numparticles

Get list Lj of neighbors for Pj

Calculate Densityj for Pj using Lj

Calculate Pressurej for Pj using Lj

Calculate acceleration Aj for Pj using Densityj and Pressurej

Move Pj using Aj and ∆t using Euler step t = t + ∆t

Cleanup all data structures

Exit
